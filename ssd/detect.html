Choose Your Method

To detect a pulsar star, you can choose from several methods based on the type of data you have. If you have numerical values in the HTRU2 format, select the numerical option. For image data in the HTRU1 format, choose the image-based detection method. If you possess both numerical values and images, you can use a combined approach for more accurate results. Additionally, if you have a PHCX file in the HTRU Medlat format, there is a specialized method tailored for this type of data. Select the appropriate method based on your available data to efficiently detect pulsar stars.
